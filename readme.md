poem
===

peomは日記を楽に書くためのCLIツールです。

なお開発中であります。

## Description

日記を毎日書く週間がない人間のために作られ、
日記を書きやすくするためのCLIツールです。

dayone2のGUIを叩くのが面倒くさい人は幸せになれるかもしれません。
